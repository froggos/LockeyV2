# Lockey

Lockey es una aplicación gestora de contraseñas de código abierto diseñada para ofrecer almacenamiento seguro, acceso rápido y control total sobre tus credenciales.

El proyecto nace con el objetivo de combinar simplicidad, seguridad y transparencia, permitiendo a los usuarios gestionar sus contraseñas sin depender de soluciones cerradas.

---

## Características

- Almacenamiento cifrado de contraseñas  
- Generador de contraseñas seguras  
- Organización por categorías o etiquetas  
- Enfoque en privacidad y control local  
- Proyecto 100% open source  

---

## Objetivo

Crear una alternativa ligera, segura y auditable para la gestión de credenciales personales.
